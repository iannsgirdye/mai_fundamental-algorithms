# Работа №1, задача №5

На вход программе, через аргументы командной строки, подается флаг и путь к файлу.
Флаг определяет действие с входным файлом. Флаг начинается с символа `'-'` или `'/'`.
Если флаг содержит в качестве второго символа опциональный символ `'n'`
(то есть `"-nd"`, `"/nd"`, `"-ni"`, `"/ni"`, `"-ns"`, `"/ns"`, `"-na"`, `"/na"`),
то путь к выходному файлу является третьим аргументом командной строки; 
иначе имя выходного файла генерируется приписыванием к имени входного файла префикса `"out_"`.
Вывод программы должен транслироваться в выходной файл. Программа распознает следующие флаги:
1. `-d` необходимо исключить символы арабских цифр из входного файла;
2. `-i` для каждой строки входного файла в выходной файл необходимо записать
сколько раз в этой строке встречаются символы букв латинского алфавита;
3. `-s` для каждой строки входного файла в выходной файл необходимо
записать сколько раз в этой строке встречаются символы, отличные от
символов букв латинского алфавита, символов арабских цифр и символа
пробела;
4. `-a` необходимо заменить символы, отличные от символов цифр, ASCII кодом,
записанным в системе счисления с основанием `16`.
