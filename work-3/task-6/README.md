# Работа №3, задача №6

Реализуйте функцию проверки правильности расстановки скобок в строке. 
На вход подаётся строка, содержащая любые символы (буквы, цифры, пробелы, знаки препинания и т.д.).
Проверке подлежат только круглые `()`, квадратные `[]`, фигурные `{}` и угловые `<>` скобки.
Программа должна определить, является ли строка корректно сбалансированной по скобкам
(каждая открывающая скобка должна иметь соответствующую закрывающую).

``` c
int check_brackets(const char *str);
```
