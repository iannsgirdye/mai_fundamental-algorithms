# Работа №3, задача №1

Реализуйте функцию перевода числа из десятичной системы счисления в систему счисления с основанием $2^r$,
$r = 1, ..., 5$. При реализации функции разрешается использовать битовые операции и операции обращения к памяти, 
запрещается использовать стандартные арифметические операции. Продемонстрируйте работу реализованной функции


## Алгоритм
Числа в компьютере хранятся в двоичной записи, поэтому можно рассмотреть алгоритм перевода числа из двоичной в $2^n$ систему счисления. Для этого нужно разделить справа налево число в двоичном представлении на группы по `n` цифр (заполнить недостающие цифры в левой группе нулями), которые будут являться двоичными представлениями **цифр** записи числа в $2^n$ системе счисления.

Получать очередную группу можно с помощью побитовой конъюнкции числа и системы счисления $2^n - 1$ (являющейся маской), а затем производить сдвиг на $n$ бит. И так до тех пор, пока все значимые биты не будут сдвинуты, то есть исходное число не станет равно нулю.


## Пример работы алгоритма
Переводим `359` в четверичную систему счисления:
- в двоичной системе счисления: `101100111`;
- разбиваем на группы: `01`|`01`|`10`|`01`|`11`;
- представляем **числа** двоичной системы счисления **цифрами** четверичной: `1`|`1`|`2`|`1`|`3`.

Итог: `11213`
