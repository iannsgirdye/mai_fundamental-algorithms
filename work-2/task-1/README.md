# Работа №2, задача №1

Реализуйте функцию с переменным числом аргументов, определяющую для каждой из переданных десятичных дробей (в виде значения вещественного типа в диапазоне `(0;1)`) имеет ли она в системе счисления с переданным как параметр функции основанием конечное представление.

Продемонстрируйте работу функции.


## Алгоритм
Нужно представить десятичную дробь в виде несократимой дроби. Если у её знаменателя все простые делители есть среди простых делителей основания нужной системы счисления, то ответ `true`, иначе `false`.
